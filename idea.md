# EduAI-Bridge

## Problem
Üniversite öğrencileri teorik bilgiye sahip olsa da sektörün beklediği pratik becerileri geliştirmekte zorlanmaktadır.

## Solution
EduAI-Bridge, öğrencilerin mevcut yeteneklerini analiz ederek hedef kariyerleri için eksik hard ve soft skill’leri belirleyen yapay zeka destekli mobil uygulamadır.

## Target Audience
- Üniversite öğrencileri
- Yeni mezunlar
- Kariyer değiştirmek isteyen bireyler

## Main Features
- Skill analysis
- AI-based career roadmap
- Personalized learning recommendations
- Progress tracking
